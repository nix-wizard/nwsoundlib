# libnwsnd

C library to read NintendoWare sound formats.

Currently able to map a CTR sound archive (.bcsar) to a struct

Planned:
- Support RVL (Wii) & Cafe (Wii U/Switch), possibly Nitro (DS) if the format is similar enough
- Read nested file types (seq, bnk, wsd, war, etc)

Extraction, conversion, and sequence synthesis will be done in separate projects using this library once the above is done

# Dependencies

c library :\)
